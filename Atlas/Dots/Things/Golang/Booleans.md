Defined by predeclared boolean type `bool`
- values `true` or `false`

Support three logical operator

| Operator  | Means                                    |
| --------- | ---------------------------------------- |
| && (and)  | true > if both statements is true        |
| \|\| (or) | true > if at least one statement is true |
| ! (not)   | true > if statement is false             |
