- number with zero ore more digits behind decimal separator, e.g. `-2.4`,  `0.1`, `3.14`, `16.984025`, and `1024.0`
- two floating-point types:
	- `float32` : 32 bits (~6-9 digits precision)
	- `float64`: 64 bits (~15-17 digits precision) > default floating-point type
