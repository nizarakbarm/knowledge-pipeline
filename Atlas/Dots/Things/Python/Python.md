- Dynamic and strongly typed programming language
- **duck typing** : the type of the class of an object is less important than the methd it defines, what the object can really do rather than what the object is
- And also **gradual typing** via **type hints** (could use dynamic or static typing with type hints)
- Everything is an **object** (numbers, strings, lists, and functions)
