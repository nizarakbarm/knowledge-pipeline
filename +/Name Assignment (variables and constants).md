- bind **names** (variables) to any type of object with assignment (=) operator:
```Python
<name> = <value>
```
- can be reassigned (rebound) to different values (object types)
#### Constants
- names too
- assigned once in a program
- defined at **module** (file) level
- visible to all functions and classes in the program
- cannot be re-assigned
